# dvidHelper

A bunch of scripts to make tinkering with DVID easier

## Features

- Automate flashing of individual DVID challenges
- Easy-to-use command-line interface
- Simplifies repetitive tasks for CTF practice or IoT security testing

## Requirements
- avrdude (preferably latest version)
## Getting Started

Clone the repository:
```bash
git clone https://github.com/MathisFranel/dvidHelper
cd dvidHelper
```
Check out the available scripts and instructions in each file to get started.
